🎯 KEY FEATURES
───────────────────────────────────────────────────────────────────

📝 GUIDED WORKFLOW
   • Select document from cache
   • Type editing request
   • AI analyzes intent
   • Automatic verification
   • Apply with confirmation

✅ AUTOMATIC VERIFICATION
   • Web research via Perplexity AI
   • Confidence scoring (0-100%)
   • Three tier system:
     - High (≥80%): Recommend adding
     - Medium (50-79%): Request source
     - Low (<50%): Reject with guidance

📎 SOURCE UPLOAD
   • URLs - Fetch and parse (✓ Working)
   • Text - Direct paste (✓ Working)
   • PDFs - Parse documents (⏳ Planned)
   • Images - OCR extraction (⏳ Planned)

💰 TOKEN OPTIMIZED
   • ~800-1600 tokens per edit
   • ~$0.001-0.003 per edit
   • 10-50x cheaper than generic chat
   • No wasted tokens on conversation

🎓 HOW TO USE

1. Start the app:
   $ cd /home/eanhd/projects/vehicles
   $ source venv/bin/activate
   $ streamlit run app.py

2. Navigate to: 💬 AI Assistant

3. Select a document from dropdown
