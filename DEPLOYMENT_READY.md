## 📊 Technical Details

### CSS Breakpoints
- **Mobile**: < 768px (primary focus)
- **Small Mobile**: < 480px (extra optimization)
- **Tablet**: 768px - 1024px
- **Desktop**: > 1024px

### Touch Target Sizes
- **Minimum**: 44x44px (iOS standard)
- **Buttons**: Full width on mobile for easy tapping
- **Form Controls**: 16px font size to prevent iOS zoom
