## 🚀 Quick Start

### Access the App
```
http://localhost:8501
```

### Generate a Document (with new styling)
1. Select vehicle (Make → Model → Year)
2. Choose service
3. Optional: ✅ Enable AI diagrams
4. Click Generate
5. Beautiful document in 10-40 seconds!

### Delete a Cached Document
1. Go to "📚 Browse Cache"
2. Select document
3. Click "🗑️ Delete"
4. Confirm
5. Done!
